## Clone the repo
```shell
git clone git@github.com:chunkingz/linkedInEasyApplyBot.git
```

## Initialize the app
```shell
cd linkedInEasyApplyBot && npm i
```

## Add .env
```shell
touch .env
```

## Add your Linkedin credentials to the env file
```shell
EMAIL=your_email@example.com
PASSWORD=your_password
```

## Run the app
```shell
npm start
```
